# SeoulSubwayMap
  공개된 서울 지하철 노선도(SVG)를 활용해 웹 브라우저에서 ZOOM, PAN이 가능한 예제
  (Hammmer.js 를 활용하여 모바일 지원 가능)
  지하철 노선 텍스트를 클릭하여 이벤트 처리 가능


# 활용한 오픈소스
 - jQuery
 - SVGPan : https://github.com/ariutta/svg-pan-zoom
